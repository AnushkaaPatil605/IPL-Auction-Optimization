# IPL Auction Optimization 

## Business Problem

IPL franchises have a limited auction budget and must maximize team performance while controlling costs. The objective of this project is to identify high-performing, cost-effective players and uncover hidden gems using data analytics.

## Project Workflow

### 1. Data Collection

Collected IPL player auction and performance datasets covering batsmen, bowlers, all-rounders, and wicket-keepers.

### 2. Data Cleaning

* Removed unnecessary columns
* Handled missing values
* Standardized player categories
* Created a unified master dataset

### 3. Feature Engineering

Developed a custom Performance Score using normalized batting and bowling metrics.

### 4. Exploratory Data Analysis

Analyzed:

* Top performers
* Cost-efficient players
* Role-wise performance
* Expensive players
* Hidden gems

### 5. Dashboard Development

Built an interactive Power BI dashboard with:

* KPI Cards
* Scatter Plot Analysis
* Hidden Gems Detection
* Value Pick Identification
* Role-wise Performance Comparison
* Auction Cost Analysis

## Key Findings

* Several low-cost players deliver performance comparable to premium players.
* Hidden gems provide excellent value under budget constraints.
* Auction price and player performance are not always strongly correlated.
* Data-driven player selection can improve auction efficiency.
